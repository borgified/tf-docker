use terraform to set up docker containers (similar to docker-compose)

tested on macOS

1. get [terraform](https://www.terraform.io/downloads.html)
2. clone this repo `git clone https://github.com/borgified/tf-docker.git`
3. cd tf-docker
4. tf init
5. tf plan
6. tf apply
